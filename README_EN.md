# ComposeDiary
A simple diary app build by Jetpack Compose, use navigation library for single activity implementation

## Tech Stack & Features
* Pure Jetpack Compose UI
* Support Add/Delete/Edit/Search diaries
* MVVM Architecture
* Kotlin Coroutine/Room/Navigation
* Dark Mode
