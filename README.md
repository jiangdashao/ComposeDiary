# ComposeDiary
基于Jetpack Compose开发的日记APP, 采用navigation库进行单activity实现

## 技术栈和功能
* 纯 Jetpack Compose 构建界面   
* 支持 添加/删除/编辑/搜索 日记
* MVVM架构
* Kotlin协程/Room/Navigation
* 支持夜间模式(自动切换)
